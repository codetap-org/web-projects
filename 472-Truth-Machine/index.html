Truthy & Falsy — The Short Version

  • Falsy values — only 7 in JS: false, 0, -0, 0n, "", null, undefined, NaN
  • Truthy — everything else, including "0", "false", [], {}
  • !!value — the double-bang trick to force any value into a real boolean
  • CSS has no booleans, but :has(), :checked, and :not() fake it beautifully
